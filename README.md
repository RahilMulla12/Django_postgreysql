# 📌 Django PostgreSQL Project (Deployed on Vercel)

This project is a web application built using **Django** with **PostgreSQL** as the database and deployed on **Vercel**. It demonstrates how to create a scalable backend using Django and host it on a modern deployment platform.
The project name is FALCON WINDOWS which is create on basis of a real world firm project used by an Enterprise
---

## 🚀 Features

- Built with Django framework
- PostgreSQL database integration
- Structured and scalable project architecture
- Deployment on Vercel
- Easy to configure and extend

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Database:** PostgreSQL
- **Deployment:** Vercel
- **Language:** Python 3
- **Frontend:** Html ,Css ,javascript
